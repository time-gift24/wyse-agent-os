# Lovable 设计系统参考

> 本文由 `npx getdesign@latest add lovable` 生成后中文化，仅作为视觉参考。Stratum 的权威规则见上级目录 `DESIGN.md`。

## 1. 氛围

Lovable 通过克制建立温度。页面使用奶油色 `#f7f4ed`，以炭黑色 `#1c1c1c` 提供清晰对比。其层级主要由同一炭黑色的不同透明度形成，因此比随意混用多种灰色更统一。细边框、紧凑标题和浅层纵深让界面接近一本制作精良的笔记本，而不是冷白色开发者工具。

原设计使用 Camera Plain Variable。Stratum 不引入该字体，只参考其字级、行高、字距和两档字重。

## 2. 配色

- 奶油色 `#f7f4ed`：页面与卡片基础。
- 炭黑色 `#1c1c1c`：主文本、标题和深色操作。
- 米白色 `#fcfbf8`：深色操作文字和高对比强调。
- 柔和灰色 `#5f5f5d`：辅助文案、说明和标签。
- 炭黑 83% / 82%：较强次级文本与正文。
- 炭黑 40%：交互边框。
- 炭黑 4% / 3%：轻微悬停和表面层级。
- 浅奶油色 `#eceae4`：静态边框、分隔线与图片轮廓。
- 蓝色焦点 `rgba(59, 130, 246, 0.5)`：键盘焦点环。

产品灰阶应尽量从炭黑色透明度派生，不混入无关色相。

## 4. 组件

### 按钮

- 主按钮：`#1c1c1c` 背景、`#fcfbf8` 文字、8px 16px 内边距、6px 圆角。
- 次按钮：透明背景、炭黑文字、`rgba(28, 28, 28, 0.4)` 边框、6px 圆角。
- 表面按钮：奶油色背景、炭黑文字、6px 圆角。
- 全圆角只用于图标操作、切换和行为上合理的 action pill。
- 激活态可降低到 0.8 不透明度；焦点必须保留清晰蓝色焦点环。

### 卡片与输入

- 普通卡片使用奶油色背景、`#eceae4` 细边框、12px 圆角，默认无阴影。
- 紧凑容器可用 8px，大型容器最大 16px。
- 输入使用奶油色背景、炭黑文字、6px 圆角和清晰焦点环；占位符使用 `#5f5f5d`。

### 导航与链接

- 导航使用奶油色表面，链接为 14–16px、400 字重。
- 普通链接依靠下划线表达交互，不通过饱和色变化制造强调。

## 5. 间距与布局

基础单位为 8px，常用间距为 8、12、16、24、32、40、56、80、96px。卡片内部使用 12–24px；页面段落之间可以更宽松。内容容器居中，展示页面最大宽度约 1200px。

Hero 字号在桌面、平板、移动端依次为 60、48、36px。移动端改为单列并缩小页面留白；触控操作需要足够尺寸。

## 6. 纵深

- Level 0：无阴影的页面平面。
- Level 1：`1px solid #eceae4` 的边框容器。
- Level 2：仅主操作可使用极轻的内嵌高光。
- Level 3：浮层或焦点可使用柔和且克制的阴影。
- Accessibility：输入和操作使用 `rgba(59, 130, 246, 0.5)` 的 2px 焦点环。

边框是主要容纳机制，阴影不用于让普通卡片悬浮。

## 7. 使用原则

应当：

- 始终以暖奶油色而不是纯白作为浅色基础。
- 以炭黑透明度构建统一灰阶。
- 用边框而不是厚阴影区分卡片。
- 大标题采用紧字距，正文保持正常字距。
- 常规层级只用 400 和 600。

避免：

- 饱和强调色、外发光和普通卡片厚阴影。
- 在矩形按钮上滥用全圆角。
- 混用多套边框逻辑。
- 使用 700 字重或扩大标题字距。
- 用装饰性渐变替代清晰的信息层级。

## 8. Stratum 采纳边界

Stratum 采纳暖中性配色、透明度层级、60/48/36/20/18/16/14px 字阶、两档字重、6/12/16px 圆角和边框主导纵深。不采纳 Camera Plain、营销页大段留白、装饰渐变和泛用胶囊。现有 Stratum 多色值仅保留给品牌标识和后续有目的的特效。
