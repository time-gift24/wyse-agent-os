## ADDED Requirements

### Requirement: Tavily 衍生产品色
Stratum Web SHALL（必须）使用从本地 Tavily 复刻站点提炼的暖中性画布、深色文本、产品蓝与语义状态色构成产品界面，并保持明暗主题中的职责一致。

#### Scenario: 浅色主题
- **WHEN** 用户在浅色主题查看任一产品路由
- **THEN** 主画布必须接近 `#fefcf5`，侧栏或轻强调表面必须接近 `#fdf8ec`，主文本必须接近 `#1f1e1e`，当前选择与焦点必须使用 `#2677ff` 产品蓝，白字填充操作必须使用同色相且对比度不低于 4.5:1 的深色值

#### Scenario: 暗色主题
- **WHEN** 用户在暗色主题查看任一产品路由
- **THEN** 界面必须从暖炭黑、暖中性表面与奶油色文本派生对应角色，并保留产品蓝的操作职责，不得使用蓝灰、紫灰、霓虹或发光表面替代层级

#### Scenario: 状态颜色
- **WHEN** 界面表达连接就绪、成功、警告、审批或错误
- **THEN** 状态必须使用可访问的语义色，绿色不得承担普通主要操作或装饰性强调职责

### Requirement: 紧凑且一致的排版
Stratum Web MUST（必须）使用现有已授权字体资源复现 Tavily 参考的信息密度，并为中英文应用统一语义字阶。

#### Scenario: 工作台主标题
- **WHEN** 概览页主标题在桌面视口渲染
- **THEN** 字号必须不超过约 44px，并使用约 600 字重、紧凑行高与轻微负字距，不得继续使用 60px 营销 Hero 字阶

#### Scenario: 正文与控件
- **WHEN** 渲染对话正文、工作台正文、导航、说明文字或紧凑控件
- **THEN** 正文与对话必须不低于 16px，导航和紧凑控件通常使用 14px，常规字重必须使用 400，强调字重必须使用 600

#### Scenario: 字体授权
- **WHEN** 实施本次视觉变更
- **THEN** 不得从参考项目复制缺少明确授权证明的字体文件，并必须继续使用项目现有字体与可靠系统回退

### Requirement: 克制的表面与形状
Stratum Web SHALL（必须）使用低对比边界、极轻纵深和 8–16px 的明确圆角层级组织信息，不依赖装饰性效果制造层次。

#### Scenario: 产品壳层表面
- **WHEN** 渲染侧栏、顶部状态区或标准内容区块
- **THEN** 这些表面必须使用细边框、暖中性背景和无阴影或极轻阴影，圆角必须根据尺寸保持在约 8–16px

#### Scenario: 禁止装饰性效果
- **WHEN** 任一产品路由完成渲染
- **THEN** 不得出现霓虹、外发光、高饱和渐变、全局颗粒、玻璃拟态、厚重投影、卡片套卡片或无信息目的的背景动效

### Requirement: 图标库统一图形操作
所有图形操作 MUST（必须）使用项目现有图标库或受保护组件已暴露的图标 API，并保持一致尺寸与视觉线宽。

#### Scenario: 导航与下拉操作
- **WHEN** 渲染菜单、下拉、关闭、发送、返回或滚动到底部操作
- **THEN** 必须使用图标库组件，不得使用 `↓`、`⌄`、`×`、箭头文本或手绘 SVG 字符模拟图标

#### Scenario: 无障碍名称
- **WHEN** 图标按钮没有可见文字标签
- **THEN** 控件必须提供可本地化的无障碍名称、清晰焦点状态和至少 44px 的触控目标

### Requirement: 有目的且可减少的动效
Stratum 自有界面动效 SHALL（必须）只表达交互或状态变化，并为减少动态效果的用户提供完整最终状态。

#### Scenario: 普通交互
- **WHEN** 用户打开抽屉、切换路由、选择导航项或触发状态反馈
- **THEN** 动效通常必须在 150–250ms 内完成，并优先只改变 `transform` 与 `opacity`

#### Scenario: 减少动态效果
- **WHEN** 用户启用 `prefers-reduced-motion: reduce`
- **THEN** 动效必须立即或近乎立即完成，内容、焦点和状态不得丢失

### Requirement: 参考内容边界
Stratum Web MUST（必须）只借鉴 Tavily 复刻站点的视觉与交互语言，不得复制其品牌或虚构其业务信息。

#### Scenario: 视觉迁移完成
- **WHEN** 用户查看改造后的 Stratum Web
- **THEN** 界面可以体现参考中的工作台布局、暖色画布、产品蓝与紧凑密度，但不得出现 Tavily 品牌、Logo、专属插图、业务文案、额度或虚构指标
